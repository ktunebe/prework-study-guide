HTML

  • The head element contains information about the webpage
    • This information is not viewable to the user
    • It contains metadata, which contains information about accessibility, search engines and performance
    • Metadata tags can include:
      • Title: defines the title of a webpage. Important for search engine optimization (SEO).
      • Style: defines the CSS styles associated with a website
      • Base: defines the base URL for a webpage
      • Link: Connects an external resource to the HTML documennt
      • Meta: defines metadata such as the character set, description, keywords, author and viewport
  • The body element represents the visible content shown to the user
    • Opening and Closing Tags
      • Opening Tags contain tag names inside of angled brackets
      • Closing Tags contain tag names preceded by a forward slash inside of angled brackets
    • Headers
      • Headers are organized by level with "h" and a number (h1, h2, h3, etc.) as their tag
      • On this page, "Prework Study Guide" is h1 and "HTML", "CSS", "Git", and "JavaScript" are all on h2 level
      • Each page should only have *1* h1
    • Lists
      1. Unordered lists contain bullet points and can be inserted with "ul" tags.
      2. Ordered lists (like this one) contain numbered points and can be inserted with "ol" tags.
      3. Items can be added to either ordered or unordered lists with "li" tags.
      4. Indent points under listed items by adding a new list within that list.
    • Paragraphs
      • Paragraphs can be added with "p"
