HTML

  • The head element contains information about the webpage
    • This information is not viewable to the user
    • It contains metadata, which contains information about accessibility, search engines and performance
    • Metadata tags can include:
      • Title: defines the title of a webpage. Important for search engine optimization (SEO).
      • Style: defines the CSS styles associated with a website
      • Base: defines the base URL for a webpage
      • Link: connects an external resource to the HTML documennt
      • Meta: defines metadata such as the character set, description, keywords, author and viewport
  • The body element represents the visible content shown to the user
    • Opening and Closing Tags
      • Opening Tags contain tag names inside of angled brackets
      • Closing Tags contain tag names preceded by a forward slash inside of angled brackets
    • Headers
      • Headers are organized by level with "h" and a number (h1, h2, h3, etc.) as their tag
      • On this page, "Prework Study Guide" is h1 and "HTML", "CSS", "Git", and "JavaScript" are all on h2 level
      • Each page should only have *1* h1
    • Lists
      1. Unordered lists contain bullet points and can be inserted with "ul" tags.
      2. Ordered lists (like this one) contain numbered points and can be inserted with "ol" tags.
      3. Items can be added to either ordered or unordered lists with "li" tags.
      4. Indent points under listed items by adding a new list within that list.
    • Paragraphs
      • Paragraphs can be added with "p"

CSS

  • Ways to style with CSS
    • Inline CSS: Applies style to a particular page element only
    • Internal Style Sheet: embeds a style sheet directly into the HTML file
    • External Style Sheet: creates a separate file for the style sheet to be linked to the HTML file (usually most ideal method)
  • Class Attributes
    • Allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
    • A period before the name you choose for a class designates it as a class selector. Any HTML element assigned to the class will gain the designated property elements from the CSS
  • Margins
    • Margins indicate how much space we want around the outside of an element
  • Padding
    • Padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git pull origin main: pulls base branch into local branch to ensure it is up to date
  • git checkout: switch between branches
    • git checkout -b branch-name: creates a new branch and switches to it
    • git checkout main: switches to main branch
    • Remember to use git pull origin main to ensure your current local branch is up to date.
  • git add -A: stages changes to be committed
  • git commit -m "custom message here": commits changes locally
  • git push origin name-of-your-new-branch: pushes up your changes to the remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable used to hold a group of data.
  • Control flow is the order in which a computer executes code in a script. JavaScript reads code in order from top to bottom.
  • A function is a set of instructions that tells the computer how to perform a certain task.
    • Functions must first be defined by assigning the task it is to complete.
    • Functions must then be called in order to complete their assigned task.
